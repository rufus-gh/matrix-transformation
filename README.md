# Matrix Transformation Visualiser

This is an html canvas matrix transformation visualiser. It allows you to type in a matrix and it will apply that transformation to the grid to help people get a visual understanding of matrix transformations. You can also apply rotations to it and see how they correspond to matrix transformations. You can try it out [here](https://matrix-transformation-visualiser.vercel.app).
